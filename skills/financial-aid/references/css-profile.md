# CSS Profile — the second form, for the biggest grants

The CSS Profile (https://cssprofile.collegeboard.org) is College Board's aid
application, used by several hundred colleges — mostly private — to award
**their own institutional grants**, which at well-endowed colleges dwarf
anything federal. FAFSA gets you federal and state money; the Profile gets you
the college's money. A family that files only the FAFSA at a Profile school
has silently declined its largest potential grant.

It opens **October 1**, the same day as the FAFSA, for the following fall.

## Which colleges require it

Do not answer from memory or any static list. Use the **participating-school
search on cssprofile.collegeboard.org** — colleges join and leave, and some
require it only from ED applicants or only for certain aid programs. Then
confirm on each college's own financial aid page, which also states the
college's Profile **priority deadline** (typically aligned with the
application round: November for ED/EA, January–February for RD — verify per
college). Record each requirement and date in that college's tracker
checklist.

## Cost and fee waivers

The Profile charges a fee for the first report and a smaller fee per
additional college. **Fee waivers are automatic based on the family's
reported income** — in recent cycles families with income up to roughly
$100,000 paid nothing, and waivers also apply to students who used an SAT fee
waiver or are in foster care / wards of the court. Verify the current
threshold and terms on cssprofile.collegeboard.org before telling a family
what they'll pay.

## How it differs from the FAFSA (so families aren't surprised)

- It digs deeper: home equity, small-business and farm value, retirement
  context, medical expenses, sibling private-school tuition. Gather prior-prior
  year tax returns, W-2s, current pay stubs, and asset/mortgage statements
  before starting.
- There is space to **explain special circumstances** in your own words (job
  loss, one-time income spike, elder care). Use it — a human reads it.
- **Noncustodial parent**: many Profile colleges require a separate submission
  from a divorced/separated student's other parent. If that parent is absent
  or refuses, colleges have a documented **waiver petition** process — ask
  each college's aid office; do not just skip it.
- Each college scores the data under its own formula, so the same Profile can
  produce very different grants at different colleges.

## IDOC — only when invited

IDOC (https://cssprofile.collegeboard.org/idoc) is College Board's secure
document-upload service some Profile colleges use to collect tax returns and
verification documents.

- **Wait for the email.** College Board emails IDOC instructions *after* the
  Profile is submitted, only if a college on the list uses IDOC. There is
  nothing to upload preemptively.
- When invited, upload promptly: signed tax returns, W-2s, and whatever the
  dashboard lists per college — one upload serves all IDOC colleges on the
  list.
- **IDOC deadlines are midnight Eastern time.** A West Coast family working
  "by end of day" misses by three hours. Treat the deadline as 8:59 p.m.
  Pacific.
- An aid offer can be withheld or delayed over one missing IDOC document;
  check the dashboard until every item shows received.

## Filing order that works

1. Early October: FAFSA first (it's shorter), then the Profile.
2. Add every Profile college before its priority deadline; adding a college
   later re-triggers the per-college fee unless waived.
3. Watch email for the IDOC invitation; upload within days, not weeks.
4. Note what was submitted where in the tracker so April surprises are
   impossible.

International families: many Profile colleges use the Profile for
international applicants too, but the surrounding rules differ — read
`international-aid.md` first.
